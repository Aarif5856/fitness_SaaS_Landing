Platform Features

Core
Functions

Workout logging, nutrition tracking, and progress charting with automatic recalculations.

Auto-Adjusting Workout Plans

After each logged session, the system compares actual reps to targets. If you complete all reps, weight increases by 2.5–5 lbs next session. If you miss 2+ reps, weight stays or decreases. Volume changes weekly based on completion rate.

Calorie and Macro Calculator

Enter current weight, goal weight, and training days per week. The system calculates daily calories using Mifflin-St Jeor, then splits into protein (1g/lb target), fats (25%), and carbs (remainder). Values update when inputs change.

Progress Charts

Line graphs for weight and measurements update after each log entry. Lift PR table shows highest weight × reps for each exercise.

Device Data Import

Pulls step count, resting heart rate, and sleep hours from Apple Health, Google Fit, or Garmin Connect. Data syncs every 15 minutes when connected.

Multi-User Access

Coaches see client workout logs in read-only mode. Assign programs by selecting a template and linking it to a client account. Notes attach to individual sessions.

Developer API

REST endpoints return JSON for workouts, meals, and metrics. Authentication via OAuth 2.0. Rate limit: 1000 requests/hour per token.

Progression Logic

When Weight
Increases

Each set has a target rep range (e.g., 8–12). When you log reps at the top of the range for all sets, the system adds 2.5–5 lbs to the next session. If reps drop below minimum, weight stays the same.

Weight +5 lbs when all sets hit top of rep range
Weight unchanged when reps are within range but not at top
Weight −5 lbs after 2 consecutive sessions below minimum reps
Calorie Calculation

How Targets
Are Set

Enter age, weight, height, and goal. The system calculates TDEE using Mifflin-St Jeor, applies a deficit or surplus, then distributes calories as: protein at 1g per lb bodyweight, fats at 0.4g per lb, carbs as the remainder.

Deficit = TDEE − 500 kcal (1 lb/week target loss)
Surplus = TDEE + 300 kcal (lean gain target)
Recalculates when you update weight or activity level
